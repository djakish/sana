sana

an object-storage-native search database, in Rust

⚠ AI-assisted project, built with Claude. An educational turbopuffer-inspired clone — learn from it, don't bet on it.

  • documents live entirely in object storage — a directory, or S3 with server-enforced conditional writes
  • vector search: IVF + faithful RaBitQ quantization, SIMD kernels, SPFresh-style local index maintenance
  • full-text BM25 with rank-safe block MAXSCORE; attribute filters native to the vector index
  • one binary: CLI, HTTP service with embedded indexing worker, automatic compaction & GC, Prometheus /metrics
